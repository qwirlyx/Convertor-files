# ConvectorNew

Веб-сервис для конвертации файлов. Поддерживает работу с документами, изображениями, видео и аудио: загрузка файла, выбор целевого формата, предпросмотр/оценка результата и скачивание готового файла.

## Что умеет сервис

- конвертация изображений, документов, видео и аудио;
- работа с PDF и DOCX;
- конвертация видео в GIF;
- настройка качества и размера изображения;
- временное хранение загруженных и готовых файлов;
- авторизация пользователей;
- SSO-вход через внешний корпоративный сервис при наличии общего секрета.

## Стек

- PHP;
- MySQL;
- HTML/CSS/JavaScript;
- FFmpeg;
- LibreOffice;
- Ghostscript;
- ImageMagick/Imagick;
- Python для отдельной обработки PDF/DOCX.

## Подготовка к запуску

1. Скопировать пример конфига:

```bash
cp config.local.example.php config.local.php
```

2. Заполнить доступы к базе данных и SSO-секрет в `config.local.php`.

3. Создать таблицу пользователей:

```bash
mysql -u user -p database_name < schema.sql
```

4. Проверить, что на сервере установлены нужные утилиты:

```bash
ffmpeg -version
libreoffice --version
gs --version
```

5. Убедиться, что PHP имеет доступ на запись в папки:

- `temp_uploads/`
- `temp_preview/`
- `temp_converted/`
- `work_tmp/`
- `logs/`

## Важные файлы

- `index.php` — входная точка приложения;
- `auth.php` — авторизация;
- `convert.php` — основная логика конвертации;
- `analyze.php` — анализ файла и предпросмотр;
- `pdf_to_docx.php` — обработка PDF/DOCX;
- `db.php` — подключение к базе через `config.local.php`;
- `schema.sql` — структура таблицы пользователей.
